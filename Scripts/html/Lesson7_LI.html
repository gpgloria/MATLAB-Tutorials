Logical Indexing

Example of a problem getting only the nonnegative elements of an array

w = []; % empty array
jj = 0;
for ii = 1:length(v)
    if v(ii) >= 0
        jj = jj + 1;
        w(jj) = v(ii);
    end
end

Example Revisited

MATLAB provides a more elegant solution:

w = []; % empty array
for ii = 1:length(v)
    if v(ii) >= 0
        w = [w v(ii)];
    end
end
