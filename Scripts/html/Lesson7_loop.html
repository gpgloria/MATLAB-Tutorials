While Loops

The general form of a while loop is like an if-statement but the block is evaluated repeatedly. The control statement is a condition.

% function [n total] = possum(limit)
%     total = 0;
%     n = 0;
%     while total <= limit             % control statement (condition)
%         n = n+1;                     % body
%         total = total + n;           % body
%     end
% fprintf('sum: %d count: %d\n', total, n)
% end

function y = approx_sqrt(x)
    y = x;
    while abs(y^2 - x) > 1e-7*x
        y = (x/y + y)/2
    end
end
% For negative x, the while loop will be in infinite loop. We must be

Exercise: While loop (next smallest prime number after n)

function k = next_prime(n)
    k = n+1;
    while isprime(k) == 0
        k = k + 1;
    end
end
